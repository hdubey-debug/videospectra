# vnvideo

Spectral video understanding as a library. Von Neumann entropy, motion, anomaly, shot detection, and recurrence over a sliding window of frame embeddings — plus clip-level event detection against text prompts. Bring your own embedder.

Status: **v0.1 alpha — pre-release.** APIs are not yet stable.

See `docs/` for requirements, architecture, and modularity guides.
